# Sorting Algorithm Comparison - QuickSort vs MergeSort

> 📅 *03 June 2026, 12:53 PM*

## 📝 Summary

- **QuickSort**: Uses partition and is in-place. Average time complexity is O(N log N).
- **MergeSort**: Uses division and merging, requiring auxiliary space. Guaranteed time complexity is O(N log N).

**Key Choices**: QuickSort for its in-place nature and average performance; MergeSort for guaranteed performance.
